# 🔗 URL Shortner Service

A scalable and secure backend URL shortening service built using Node.js, Express.js, PostgreSQL, and Drizzle ORM.  
This project allows users to generate short URLs, authenticate using JWT, and manage shortened links efficiently.

---

# 🚀 Features

- 🔐 JWT Authentication & Authorization
- 👤 User Registration & Login
- ✂️ Generate Short URLs
- 🎯 Custom Short Codes Support
- 🔄 URL Redirection Logic
- 🛡️ Zod Validation
- 🗄️ PostgreSQL Database
- ⚡ Drizzle ORM Integration
- 🧱 MVC Architecture
- 🐳 Docker Support
- 🌱 Environment Variable Configuration
- 🔒 Protected Routes using Middleware
